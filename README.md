# vue-dynamic-component

vue의 native 기능인 dynamic-component입니다.
vue-router 없이 싱글 컴포넌트에서 동적으로 여러개의 컴포넌트를 라우팅 할 수 있는 예제를 만들었습니다.

```
<component v-bind:is="componentName"></component>
```
